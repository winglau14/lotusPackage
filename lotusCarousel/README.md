# lotus-carousel

> vue 滑动轮播、无缝轮播组件

## how to use 

### npm install lotus-carousel --save or yarn add lotus-carousel
### 插件的方式引入使用
1.main.js <br/>
import lotusCarousel from 'lotus-carousel' <br/>
import 'lotus-carousel/dist/lotus-carousel.min' <br/>
Vue.use(lotusCarousel) <br/>
